ABOUT US

This website is created as a part of a college project using HTML, CSS, and Bootstrap. Our goal is to build a simple, responsive, and user-friendly website that includes basic pages such as Home, Registration, Login, Blog, and About Us.

The purpose of this project is to practice front-end development and understand how modern websites are structured. We focus on clean design, easy navigation, and minimal layout so everything is clear and readable.

This project helps students learn how to convert ideas into working web pages using basic technologies without backend or database.
